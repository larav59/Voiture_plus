# Backend Superviseur industriel Voiture+

## Description

Le backend est responsable de la logique serveur du système de supervision des véhicules. Il gère les échanges entre le frontend et fait office de passerelle avec le broker MQTT en relayant les commandes de l'opérateur en messages MQTT pour le broker.

Ce backend permet également de pouvoir suivre en temps réel l'état des véhicules et leur itinéraire.
